# AI Resume Screening System

## Overview
AI-powered Resume Screening System that predicts the most suitable job role based on resume skills using Machine Learning and NLP.

## Features
- Resume skill analysis
- Job role prediction
- Streamlit web interface
- Machine Learning model

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
